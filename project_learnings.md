# Project Learnings

## Dataset
- California Housing Dataset
- 20,640 samples
- 8 features

## EDA Findings
- Median Income showed strongest correlation with house prices.
- Income distribution is right skewed.
- Geographic location significantly impacts house prices.
- Bay Area and Los Angeles regions contain expensive houses.

## Model Findings
- Linear Regression achieved moderate performance.
- Random Forest outperformed Linear Regression.
- Correlation does not always indicate feature importance.

## Important Lesson
Cross Validation provides a more reliable estimate than a single train-test split.
